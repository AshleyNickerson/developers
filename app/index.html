Get Started in 1 Minute

  1. Add your account Token

    After installing and starting Qordoba CLI, Initialize your command line and add your organization-id, project-id

  2. Add your files to Qordoba

    Simple push you files to the cloud in one simple command

  3. Pull back your final keys

    At any time you can call back your keys with content as files in one simple command

$ qordoba init  \
  --organization-id  1234 \
  --access-token  xxxxxxxxxxxxxxxxxxxxxxxxxxx \
  --project-id  4422 \
$ qordoba push keys.json \
  ->  Loading Qordoba config... \
  ->  Uploading keys.json \
  ->  Please setup new version tag:VersionTag: v1 \
  ->  Uploaded keys.json successfully
$ qordoba pull \
  ->  Starting Download the file(s) \
  ->  Downloaded file `web.json` for src `core.json`
